Selectors while doing css

Css can be written using different kinds of selectors. We can use id class and elements for applying css .

  • Element

    • The element selector can be used directly by using the name of the element and giving the styling to it.

  • Class

    • For using the class selector we can use class attribute in any elements that has multiple values and change the style of that specific element.The use of class will not make any changes in the html. By using . and class name after dot and writing the styles inside the {} braces we can use the class selector for doing the styling. DOt is known as a class selector

  • ID

    • For using the id selector we use id attribute in any elements that we want to make changes.After that by using # and id name after hastag and writing the styles inside the {} braces we can use the id selector for doing the styling. Hastag(#) is known as a id selector

  • Pseudo
    1. Hover:

      2. We can use tag/selector name: hover for applying the hover effect to a particular tag/selector
