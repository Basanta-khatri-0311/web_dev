  1. Cascade

    2. In css the styles is applied from top to bottom i.e one after another.

  2. Specificity

    3. If there are two or more CSS rules that point to the same element, the selector with the highest specificity value will "win", and its style declaration will be applied to that HTML element. By hovering over them in the css file we can se the specificity of that particular element.

    The specificity will be first checked and then the cascade rule will be applied. So the specificity for selectors will be element < class < id., i.e the specificity of id will be higher then everyone and then class and then the element.

    The addition or combination of the selectors will increase the specificity.

  3. Inheritance

    4. The inherit keyword specifies that a property should inherit its value from its parent element.

    The inherit keyword can be used for any CSS property, and on any HTML element.
