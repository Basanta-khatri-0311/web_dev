Data Types in JavaScript

There are two types of data types in JavaScript.

  1. Primitive (Value Types)

  2. Non Primitive (Reference Types)

In this part we'll learn about primitive data types.

We can classify primitive data types in 7 categories.

  1. string
  2. number
  3. boolean
  4. undefined
  5. null
  6. bigint
  7. symbol

There is only one type of non primitive data types that is called Object.

Objects are further divided into three categories.

  • Object Literals
  • Arrays
  • Functions
