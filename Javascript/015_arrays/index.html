Arrays in JavaScript

Basic Array Methods

  1. Pop()

    2. Removes an element from last.

  2. push()

    3. Adds ann element at last

  3. shift()

    4. Remove elements from front

    changes/shifts the index of the element

  4. unshift()

    5. Add the number from front

  5. concat()

    6. It creates a new array by concatenating the arrays.
    The original arrays wont be changed.
    Can concat multiple arrays

  6. indexof()

    7. Gives the indexing of the element, and gives -1 for element which is not in the array.
    Gives 1st index of the element if the array contains duplicate values. Its case sensitive.

  7. includes()

    8. Gives boolean result if the element is in the array or not.

  8. reverse()

    9. Reverses the order of the elements and their indexing as well.
    Changes the original array into reverse. will not give a new array

  9. sort()

    10. Sorts the elements in alphabetical order as in default. Modifies the original array. For default use must be in same order

  10. slice()

    11. Makes the new array without changing actual array and gives the sliced value. Start value is included and end value is excluded.

  11. splice()

    12. Same as slice but changes the default array. Can add and remove in the default array
